# Customer Support Performance Analysis

**Owner:** Akash Niranjappa  

## Overview
This project analyzes customer support interaction data to evaluate
key performance metrics such as Customer Satisfaction (CSAT),
Average Handle Time (AHT), and First Call Resolution (FCR).

## Tools Used
- Excel (Power Query, Pivot Tables)
- SQL
- Power BI

## Key Insights
- Identified trends impacting customer satisfaction
- Analyzed agent-level performance metrics
- Built KPI dashboards for operational monitoring

## Outcome
Provided data-driven recommendations to improve support efficiency
and customer experience.
